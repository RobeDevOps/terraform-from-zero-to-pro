# Video 002 - Clean Structure

En este video:
- Crear multiples ficheros para organizar mejor.
- Delegar a cada fichero lo que corresponde como tal.
- Ayudar a identificar los elementos en desarrollo.

Desde una mirada inicial puedes identificar aquellos recursos con los que trabaja el modulo y en donde hacer los cambios pertinentes.
Creeme, he estado en proyectos donde tienes que navegar hasta una linea y luego hacer los cambios. Si alguien mas en el equipo require manejar el codigo contigo, es todo un proceso doloroso.

Commandos utilizados:
```
terraform init
terraform plan
terraform apply -auto-approve
```

# Recuerda limpiar tu environment
```
terraform destroy -auto-approve
```
